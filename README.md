# Excel-sales-dashboard
Interactive Sales Performance Dashboard using Microsoft Excel
Project Overview 
This project demonstrates how Microsoft Excel can be used to clean, analyze, and 
visualize business data. 
Using a dataset of 5,000 e-commerce sales transactions, I developed an 
interactive dashboard that provides insights into sales performance, customer 
satisfaction, regional trends, and payment methods. 
Business Problem 
Management required a dashboard to monitor sales performance and identify key 
business trends. The raw dataset contained missing values, duplicate records, 
inconsistent category names, and invalid entries that needed to be cleaned before 
analysis. 
Objectives 
• Clean the dataset  
• Perform exploratory data analysis  
• Create Pivot Tables  
• Build KPI metrics  
• Design an interactive dashboard  
• Present actionable business insights  
Tools Used 
• Microsoft Excel  
• Pivot Tables  
• Pivot Charts  
• XLOOKUP  
• Conditional Formatting  
• Slicers  
Key Insights 
• Identified the highest-performing sales region.  
• Determined the best-selling product category.  
• Compared payment method usage.  
• Evaluated customer satisfaction across regions.  
• Analysed monthly sales trends.
